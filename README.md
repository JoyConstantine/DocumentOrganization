# 文件整理工具

一个Windows下的文件整理工具，可以帮助你快速归类电脑上的文件，给文件做标签分类，并按照分类需求整理到指定目录。

## 功能特性

- **文件扫描**：支持扫描指定目录下的所有文件，包括子目录
- **文件标签**：可以为文件添加自定义标签，方便分类管理
- **文件整理**：根据标签将文件移动到对应分类目录
- **多种分类方式**：支持按扩展名、大小、日期等自动分类
- **文件名冲突处理**：自动处理文件名冲突，生成唯一文件名
- **友好的用户界面**：使用Tkinter实现的图形界面，操作简单直观

## 系统要求

- Windows操作系统
- Python 3.6+

## 安装与运行

### 方法一：直接运行（需要Python环境）

1. 确保已安装Python 3.6或更高版本
2. 下载或克隆本项目到本地
3. 打开命令行，进入项目目录
4. 运行以下命令启动工具：

   ```bash
   python main.py
   ```

### 方法二：打包为可执行文件（无需Python环境）

可以使用PyInstaller将项目打包为可执行文件：

1. 安装PyInstaller：

   ```bash
   pip install pyinstaller
   ```

2. 打包项目：

   ```bash
   pyinstaller --onefile --windowed main.py
   ```

3. 在`dist`目录中找到生成的可执行文件

## 使用方法

1. **选择源目录**：点击"浏览"按钮选择要整理的文件所在目录
2. **扫描文件**：点击"扫描"按钮扫描目录下的所有文件
3. **添加标签**：在文件列表中选择文件，输入标签名称，点击"添加标签"
4. **选择目标目录**：点击"浏览"按钮选择整理后文件的存放目录
5. **整理文件**：选择要整理的文件，点击"整理文件"按钮开始整理

## 自动分类功能

除了手动添加标签外，工具还支持以下自动分类方式：

- **按扩展名分类**：根据文件扩展名自动分类
- **按大小分类**：根据文件大小自动分类
- **按日期分类**：根据文件修改日期自动分类

## 注意事项

1. 整理文件时会移动原始文件，请确保已备份重要文件
2. 大型目录扫描可能需要一些时间，请耐心等待
3. 如果遇到权限问题，请以管理员身份运行工具

## 示例

### 手动标签分类

1. 扫描包含各种文件的下载目录
2. 为图片文件添加"图片"标签
3. 为文档文件添加"文档"标签
4. 为视频文件添加"视频"标签
5. 选择目标目录并点击"整理文件"
6. 工具会自动在目标目录创建"图片"、"文档"、"视频"子目录，并将对应文件移动到相应目录

### 自动分类

1. 使用`organize_by_extension`方法按扩展名自动分类
2. 使用`organize_by_size`方法按文件大小自动分类
3. 使用`organize_by_date`方法按修改日期自动分类

## 开发说明

项目结构：

- `main.py`：主入口文件
- `ui.py`：用户界面模块
- `scanner.py`：文件扫描模块
- `organizer.py`：文件整理模块
